# mobile_gnp
GNP React Native Mobile Application

React && React Native
```bash
React: 16.13.1
ReactNative: 0.63.4
```

Android
```bash
buildToolsVersion = "29.0.2"
minSdkVersion = 16
compileSdkVersion = 29
targetSdkVersion = 29
```

Run on Android Device
```bash
npm install
react-native run-android
```
